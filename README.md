# 🪑 Furniture Sales Web (Laravel)

Website penjualan mebel berbasis Laravel yang digunakan untuk mengelola produk dan transaksi penjualan.

## ✨ Fitur

* Login & Register User
* CRUD Produk (Tambah, Edit, Hapus)
* Sistem Transaksi Penjualan
* Dashboard Admin

## 🛠️ Tech Stack

* Laravel 12
* MySQL
* Bootstrap

## 🚀 Cara Menjalankan Project

1. Clone repository
2. Jalankan perintah:
   composer install
3. Copy file env:
   cp .env.example .env
4. Generate key:
   php artisan key:generate
5. Jalankan migration:
   php artisan migrate
6. Jalankan server:
   php artisan serve
